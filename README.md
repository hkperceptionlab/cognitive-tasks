# Cognitive Tasks — H.K. Perception Lab

A collection of web-based cognitive tasks. The first is the Stroop task, with
separate versions for youth and adults and support for four languages.

This is a **demonstration** tool, not a training program. It makes no promise of
improving cognitive ability — the point is simply to let people experience how
these classic tasks work.

**No data is collected.** Nothing is sent to any server; every record stays in
your own browser and never leaves your device.

Built by a high school student, with coding help from an AI tool (Claude Code).
