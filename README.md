# Cognitive Tasks — H.K. Perception Lab

A collection of web-based cognitive, perception, and attention tasks —
currently Stroop, Go/No-go, Corsi block-tapping, Digit Span Backward, Simple
Reaction Time, the Simon task, a Stop-signal (stopping) task, Mental Rotation,
N-back, a size discrimination-threshold (JND) task, a visual search task, a
SART (sustained attention) task, an attentional-blink task, an emotional Stroop
task, and an emotional dot-probe task — each with separate versions for youth and
adults and support for four languages. There
are also standalone perception demos (experiences, not timed tasks): a
blind-spot demo, a Necker-cube (bistable perception) demo, a negative-
afterimage demo, and an inattentional-blindness demo. A new **practice**
series adds guided experiences rather than measurements — a breath-counting
practice (Levinson 2014), a body-scan meditation (with optional voice
guidance), and a loving-kindness & gratitude meditation.

This is a **demonstration** tool, not a training program. It makes no promise of
improving cognitive ability — the point is simply to let people experience how
these classic tasks work.

**No data is collected.** Nothing is sent to any server; every record stays in
your own browser and never leaves your device.

Built by a high school student, with coding help from an AI tool (Claude Code).
